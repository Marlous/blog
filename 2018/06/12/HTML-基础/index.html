HTML 基础

一 概念

  1. HTML 文档基本构成：标签，属性，元素。

  2. Head 放置一些辅助元素（主要为样式表 <style></style>；JavaScript 脚本 <script><script>）。
    Body 放置网页主体内容。

  3. 注释在 HTML,CSS,JavaScript 中的区别：
    HTML中 <!-- --> ，
    CSS中 /* */ ，
    JavaScript 单行 // 多行 /* */ 。

  4. 使用小于号，大于号，引号等需要使用参考字符，以 & 开始，以 ; 结束。如 1<2 写成 1&lt;2 。
