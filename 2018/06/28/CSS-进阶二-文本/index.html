CSS 进阶二 文本

作者 Marlous 日期 2018-06-28
CSS
CSS 进阶二 文本

一 字体

1 字体大小

font-size

1em = 12px，百分比也以 12px 为基准。1em 取决于 px 设置的大小。

2 字体

font-family

3 加粗

font-weight

4 斜体

font-style
数值为数字则直接继承，数值为百分比则先计算再继承。

5 多属性缩写

6 文字颜色

color
rgba() 中 a 为透明度。0 为全透明，1 为不透明。

二 对齐方式

1 对齐方式

text-align
justify 为两端对齐。

2 垂直对齐

vertical-align
百分比以行高为参照向上偏移。

3 首行缩进

text-indent
百分比以容器宽度为参照。

三 格式处理

1 white-space

white-space
设置如何处理元素内的空白。

2 长单词自动换行

word-wrap
设置为 break-word

3 任意字母间换行

word-break

四 文字修饰

1 文字阴影

text-shadow
三个 px 参数：x 轴偏移，y 轴偏移，模糊半径。

2 文字加线

text-decoration

五 高级设置

1 长文本省略

text-overflow
让长文本不显示全，后面只显示三个省略。注意要配合两个属性使用。

2 定义鼠标图形

cursor
可用于将文本通过 CSS 做成按钮。

可以是一个图片，为图片时可多用一个做备份用（以防图片不可用）；
auto 为自动变换；
default 为箭头；
none 为消失；
help 为问号；
pointer 为手形状；
zoom-in 为放大镜；
move 在移动一些元素时可以使用。

3 强制继承值

inherit
当值为 inherit 时可强制继承父元素的值。
